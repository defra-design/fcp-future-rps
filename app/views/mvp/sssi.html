SSSI consent

If you apply for SFI actions on land that's a site of special scientific interest (SSSI), you must get SSSI consent from Natural England.

You need to get this consent before you do your selected SFI actions on SSSI land. Some actions can be done without SSSI consent - this is set out in each action in the 'find funding for land or farms' tool.

Read the guidance on SSSI consent.
