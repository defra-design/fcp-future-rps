The actions looping logic is hard coded at the moment:

  • Add CMOR1 on choose actions screen
  • Then click 'remove parcel' on actions review screen
  • Then add CMOR1 again on choose actions screen
  • Then click 'add another action' on actions review screen
  • Then add UPL2 on choose actions screen
  • Then click 'change' on UPL2 on actions review screen
  • Then add UPL3 on choose actions screen
  • Then remove UPL3 on actions review screen
  • You should be back to just having CMOR1 again in list on actions review screen
