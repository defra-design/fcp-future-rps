What changed

Actions review screen - Design and Accessibility improvements

We changed the design of the actions review screen because:

  • the previous tabular design was more difficult for readibility once multiple parcels and actions were added
  • we anticipated that users with screenreaders would struggle to associate which parcels were associated with which actions on the old tabular design
  • using a summary card component solves the above problem by clearly announcing to screenreaders which parcels are associated with which actions when a new card is focused upon. Each card is also listed on the contextual screenreader menu for efficient navigation for these users.
  • the new design also helps users with both cognitive disabilities and visual impairements (eg zoom text users) by presenting information in a clearer and more structured way
  • the summary card component brings visual separation and consistency across both action and agreement level payment information, allowing users to more clearly understand how their total payment figure has been reached
